2 

3A safeguard layer ON TOP of the analyzer: it inspects the analyzed results and 

4surfaces data an operator should double-check -- a heat that looks stopped but 

5was entered without a restart, results with no 1st place, an empty scoring

24# A recorded lap FASTER than the class could physically go is a MIS-CLICK: the 

25# operator clicked the wrong boat (a spurious crossing at another boat's line time) 

26# or double-tapped when several boats cross close together. The fastest possible lap 

27# is the class's shortest lap length / its top speed (the pattern's '@<speed>' km/h 

28# hint, default DEFAULT_CLASS_SPEED); a lap under that is impossible. Per-class speed 

29# beats a fixed window -- an F-500 runs ~3x a GT-15, so a "short" lap differs by 

30# class. Reported for Edit-Records review, NEVER auto-corrected (scoring stays the 

31# analyzer's). Uses gettimes, so a mis-click the operator already disabled is 

32# absorbed into the next lap and not re-flagged. Time-trials excluded (solo runs). 

33# 

34# The opposite mis-click is a MISSED click: the operator didn't click a crossing, so 

35# one recorded lap spans two -- it reads ~2x the boat's median. This can't be decided 

36# physically (a boat CAN run slow), so it is only a POSSIBLE finding, corrected by 

37# INSERTING a mark in Edit Records. Circuit and endurance need SEPARATE handling 

38# because their lap-time distributions differ completely: a circuit boat never pits, 

39# so any lap well above typical is worth a look (unbounded); an endurance boat pits 

40# and breaks down (huge laps that are NOT missed clicks -- clear outliers that must 

41# not be allowed to shape the circuit rule), so only a lap in a narrow band around 2x 

42# counts. 

43_MISSED_CLICK_FACTOR = 1.8 # lower edge, both disciplines (~2x with margin) 

44_ENDURANCE_MISSED_MAX = 2.5 # endurance upper edge: above this it's a pit/breakdown, not a miss 

45_MISSED_CLICK_MIN_LAPS = 4 # need a few laps for a stable median

48def _laps(marks): 

49 return sum(1 for m in marks if abs(m[0]) in (LAP, INSERTED_LAP) and m[0] > 0)

52# The "restart required" lap fraction is a DISCIPLINE rule, not a fixed constant: 

53# U.I.M. 311.02.1 uses 0.70, but non-UIM competitions may differ. Read it from 

54# event config (defaulting to the UIM value) so it can be exposed/edited per event 

55# in the Rules tab. NOTE: the analyzer still hardcodes requiredlapscoef=0.70; the 

56# follow-up (Rules-tab exposure) should thread this same value into analyze() so 

57# the two agree. Keep the default in sync with the analyzer until then. 

58_DEFAULT_REQUIRED_LAPS_COEF = 0.70

69def check_results(eventdata): 

70 """Return a list of :class:`Finding` warnings about suspicious result data. 

71 

72 Pure and defensive: never raises (a failing check is swallowed) and never 

73 mutates ``eventdata``. Ordered by class then heat for stable display. 

74 """ 

75 findings = [] 

76 try: 

77 record = eventdata.get("record", {}) or {} 

78 ss = eventdata.get("scoringsystem", []) or [] 

79 rulecodes = rule_action_codes(eventdata) 

80 

81 if not ss and any(record.get(cl) for cl in record): 

82 findings.append(Finding("warning", None, None, "empty-scoring", 

83 "scoring system is empty — every result scores 0 points")) 

84 

85 # Per-kind dispatch runs on the phase model (PHASES.md §8 step 2): a class's 

86 # kind and pattern come from its Phase, not from re-inferring them here via the 

87 # class/heat suffix. phase_of maps each legacy class name to its Phase; the 

88 # legacy record dict is still iterated (same class-then-heat order) so findings 

89 # stay byte-identical. Skipping a time-trial/qualification *phase* replaces the 

90 # old per-heat ``h[-1] in ('q','t')`` skip -- identical on real data (a /T,/Q 

91 # class's heats all carry the suffix); a hint-kinded class with a plain heat id 

92 # can't occur in the legacy corpus. 

93 # 

94 # TEMPORARY equivalence, NOT the §4 end state: this lumps timetrial and 

95 # qualification into one skip only to stay byte-identical with legacy. Per 

96 # PHASES.md §4 they diverge -- timetrial gets a *light* (physics-only) mis-click 

97 # check, and qualification is a real mass-start kind (fast+slow mis-click). When 

98 # the §4.1 code ripples land, qualification comes OUT of this skip and timetrial 

99 # gets the light check. 

100 phase_of = class_phase_map(eventdata) 

101 

102 classes = [c for c in get_classes(eventdata) if c in record] 

103 for cl in classes: 

104 ph = phase_of.get(cl) 

105 if ph is None or ph.kind in ("timetrial", "qualification"): 

106 continue # scored differently, not click-checked 

107 heats = record.get(cl) or {} 

108 for h in sorted(heats): 

109 info, rec = heats[h] 

110 

111 # Mis-click detection: a boat whose EFFECTIVE lap (gettimes absorbs a 

112 # disabled/already-corrected click) is faster than physically possible 

113 # (or, for a missed click, ~2x its median) likely got a spurious/absent 

114 # crossing. Runs independent of analyze/placements (a heat where everyone 

115 # DNF'd can still have a click error). Report for Edit-Records review; 

116 # scoring is unchanged. Only mass-start racing -- a time-trial or 

117 # qualifier runs solo, so no pack and no wrong-boat click.

119 min_lap = _pattern_min_lap_time(ph.pattern) 

120 if min_lap: 120 ↛ 124line 120 didn't jump to line 124 because the condition on line 120 was always true

121 findings.extend(_misclick_findings( 

122 cl, h, rec, min_lap, ph.kind == "endurance")) 

123 

124 try: 

125 res = analyze(h, heats[h], ss, rulecodes) 

126 except Exception: 

127 continue # analyze errors are the analyzer's to report 

128 places = sorted(res[p]["place"] for p in res if res[p]["place"] > 0) 

129 if not places: 

130 continue 

131 

132 # A stopped fixed-lap heat only needs a restart if the leader 

133 # finished under the discipline threshold (U.I.M. 311.02.1: 70%); 

134 # and a stop already recorded via inserted-lap crossings is handled. 

135 # Flag only a genuinely missing restart, and say what to do. 

136 course_len = len(info.get("course", []) or []) 

137 if info.get("duration") is None and course_len: 

138 maxlaps = max((_laps(rec[p]) for p in rec), default=0) 

139 coef = _required_laps_coef(eventdata) 

140 minrequired = int(coef * course_len) 

141 stop_handled = any(abs(m[0]) == INSERTED_LAP for p in rec for m in rec[p]) 

142 has_restart = (h + "r") in heats or (h + "R") in heats 

143 if maxlaps < minrequired and not has_restart and not stop_handled: 

144 pct = int(coef * 100) 

145 findings.append(Finding( 

146 "warning", cl, h, "incomplete-heat", 

147 "leader completed only %d of %d laps (< %d%%) and no restart is " 

148 "recorded — the heat may have been stopped early. A boat scores only " 

149 "with a lap after the race stop line: record each finished boat's " 

150 "stop-race-line crossing (or insert a lap mark after it), or run a " 

151 "restart (required only if the leader finished under %d%% of the " 

152 "course)." % (maxlaps, course_len, pct, pct))) 

153 

154 # Scored placings should be a contiguous 1..N. Any gap (or a missing 

155 # 1st) means a boat that would place is unclassified -- commonly a 

156 # stopped heat where it has no lap after the race stop line, so it is 

157 # DNF (see the analyzer's leadertime rule). 

158 if places != list(range(1, len(places) + 1)): 

159 missing = sorted(set(range(1, len(places) + 1)) - set(places)) 

160 findings.append(Finding( 

161 "warning", cl, h, "place-gap", 

162 "scored placings are %s, not a contiguous 1..%d (missing %s) — a boat that " 

163 "would place is unclassified (commonly DNF: no lap recorded after the race " 

164 "stop line); if it finished, record its stop-line crossing" 

165 % (places[:8], len(places), missing or "n/a"))) 

166 except Exception: # validation must never break the app 

167 pass 

168 return findings

171def _pattern_min_lap_time(pattern):

174 default 150). ``None`` if the pattern is missing or unusable.""" 

175 if not pattern: 175 ↛ 176line 175 didn't jump to line 176 because the condition on line 175 was never true

176 return None 

177 try: 

178 lengths = [x for hh in crack_race_pattern(pattern)[0] for x in hh if x > 0]

185def _misclick_findings(cl, h, rec, min_lap, endurance): 

186 out = [] 

187 for pid in sorted(rec, key=str): 

188 laps = gettimes(rec[pid]) 

189 fast = [round(t, 2) for t in laps if 0 <= t < min_lap] 

190 if fast: 

191 out.append(Finding( 

192 "warning", cl, h, "misclick", 

193 "boat %s has %d lap(s) faster than physically possible for this class (< %.0fs, " 

194 "%s) — likely a mis-click (wrong boat / double tap when boats cross together); " 

195 "review and correct in Edit Records" % (pid, len(fast), min_lap, fast))) 

196 # Missed click: a lap ~2x the boat's median. Circuit (no pits) flags any lap 

197 # well above typical; endurance (pits are huge legit outliers) only a narrow band. 

198 if len(laps) >= _MISSED_CLICK_MIN_LAPS: 

199 med = statistics.median(laps) 

200 if med > 0: 200 ↛ 187line 200 didn't jump to line 187 because the condition on line 200 was always true

201 lo, hi = _MISSED_CLICK_FACTOR * med, (_ENDURANCE_MISSED_MAX * med if endurance 

202 else float("inf")) 

203 slow = [round(t, 2) for t in laps if lo <= t <= hi] 

204 if slow: 

205 out.append(Finding( 

206 "warning", cl, h, "missed-click", 

207 "boat %s has %d lap(s) roughly 2x its typical %.0fs (%s) — a lap crossing " 

208 "may have been MISSED (or the boat ran slow); if a click was missed, insert " 

209 "a mark in Edit Records" % (pid, len(slow), med, slow))) 

210 return out
